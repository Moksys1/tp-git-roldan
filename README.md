# TP5: Control de Versiones con Git
Este repositorio es parte de la Actividad N°5 para la materia Taller de Programación en el ISDEM.
El objetivo es aprender a gestionar el historial de un proyecto mediante commits y el uso de ramas en Git.
Aquí se registrará la evolución de la estructura básica del sitio, incluyendo archivos HTML y CSS.
Este cambio se realizó desde la interfaz de GitHub.
